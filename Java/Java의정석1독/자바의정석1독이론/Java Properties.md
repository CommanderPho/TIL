# Java Properties
* 내부적으로 Hashtable을 사용하며, key와 value를 (String, String)로 저장
* 주로 어플리케이션의 환경설정에 관련된 속성을 저장하는데 사용되며 파일로부터 편리하게 값을 읽고 쓸 수 있는 메서드를 제공
* 따라서 간단한 입출력은 Properties를 활용하면 몇 줄의 코드로 쉽게 해결 가능
