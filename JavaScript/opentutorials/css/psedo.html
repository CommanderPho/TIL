방문함 방문안함
  • link : 방문한 적이 없는 링크
  • visited : 방문학 적이 없는 링크
  • hover : 마우스를 롤오버 했을 때
  • active : 마우스를 클릭했을 때
  • 위의 선택자는 순서대로 지정하는 것이 좋다.
  • visited의 경우는 보안상의 이유로 아래와 같은 속성만 변경기 가능
    • color
    • background-color
    • border-color
    • outline-color
    • The color parts of the fill and stroke properties
